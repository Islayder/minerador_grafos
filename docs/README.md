# Documentação

Material de apoio ao relatório técnico e à arquitetura do sistema.

```
docs/
├── diagramas/     # modelos UML (.puml e .png)
└── relatorio/     # LaTeX, PDF final, template SBC e figuras
```

| Pasta | Conteúdo |
|-------|----------|
| [`diagramas/`](diagramas/) | Dez diagramas PlantUML da arquitetura, domínio, fluxos e API de grafos |
| [`relatorio/`](relatorio/) | Relatório oficial (`Relatório_Técnico___Teoria_dos_Grafos_e_Computabilidade__Oficial_.*`), template SBC e figuras |

O relatório descreve mineração, construção dos grafos, métricas, resultados sobre **giscus/giscus** e validação com 167 testes unitários.

## Overleaf

Para editar o LaTeX: upload do `.tex`, `sbc-template.sty` e da pasta `figuras/` no mesmo nível do arquivo principal.
