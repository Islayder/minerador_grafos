# Relatório técnico

Relatório final da disciplina Teoria dos Grafos e Computabilidade — análise de colaboração no repositório **giscus/giscus**.

| Arquivo | Descrição |
|---------|-----------|
| `Relatório_Técnico___Teoria_dos_Grafos_e_Computabilidade__Oficial_.tex` | Fonte LaTeX |
| `Relatório_Técnico___Teoria_dos_Grafos_e_Computabilidade__Oficial_.pdf` | Versão entregue (PDF) |
| `sbc-template.sty` | Template de formatação SBC |
| `figuras/` | Imagens referenciadas no documento |

## Escopo

O documento apresenta arquitetura em camadas, mineração integral do repositório, construção de quatro grafos direcionados ponderados, onze métricas topológicas calculadas sobre o grafo integrado, interfaces CLI e desktop, exportação GEXF e resultados quantitativos e visuais da rede de colaboração.

## Overleaf

Upload do `.tex` (como arquivo principal), `sbc-template.sty` e da pasta `figuras/` no mesmo diretório.

Detalhamento das figuras: [`figuras/README.md`](figuras/README.md).
